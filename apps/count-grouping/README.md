# Count Grouping

An educational game for kids to practice counting and grouping objects. Children drag objects into groups and learn multiplication concepts through visual manipulation.

## Features

- Drag-and-drop interface for grouping objects
- Visual feedback for correct/incorrect groupings
- Sporty, kid-friendly theme with warm colors
- Mobile-friendly touch interactions

## Development

```bash
npm install
npm run dev
```

## Deployment

```bash
./scripts/deploy.sh count-grouping
```
